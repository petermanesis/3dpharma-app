# Autocomplete Libraries

## Overview
This document describes the libraries used to implement autocomplete functionality in the Drug Assessment Tool.

## Library Used

### streamlit-searchbox
**Purpose:** Provides searchbox component with dropdown autocomplete for Streamlit applications

**Installation:**
```bash
pip install streamlit-searchbox
```
